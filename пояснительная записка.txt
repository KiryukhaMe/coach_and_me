ПОЯСНИТЕЛЬНАЯ ЗАПИСКА
к проекту "Coach and Me!" - система учета тренировок

1. Введение

1.1. Назначение проекта
Coach and Me! - это веб-приложение для учета персональных тренировок, которое позволяет:
- Записывать выполненные упражнения и нагрузки
- Отслеживать прогресс в достижении спортивных целей
- Хранить историю всех тренировок

1.2. Целевая аудитория
- Посетители тренажерных залов
- Люди, занимающиеся фитнесом дома
- Персональные тренеры

1.3. Актуальность
Приложение решает проблему потери данных о тренировках и сложности анализа прогресса при использовании бумажных носителей.

2. Техническое описание

2.1. Функциональные возможности
- Регистрация и вход пользователей
- Создание, редактирование и удаление тренировок
- Автоматическое сохранение даты тренировки
- Просмотр истории занятий

2.2. Используемые технологии
Backend: Python 3, Flask, SQLAlchemy
Frontend: HTML5, CSS3, Bootstrap 5
База данных: SQLite
Аутентификация: Flask-Login

2.3. Структура проекта
- Модель данных:
  * Пользователи (User)
  * Тренировки (Workout)
- API: обработка HTTP-запросов через Flask
- Интерфейс: адаптивные веб-страницы

3. Реализация

3.1. Основные функции
1. Система авторизации:
- Регистрация по email и паролю
- Шифрование паролей

2. Управление тренировками:
- Добавление новых записей
- Редактирование и удаление
- Автоматическое сохранение даты

3. История тренировок:
- Просмотр всех занятий
- Сортировка по дате

3.2. Пример реализации
@app.route('/add_workout', methods=['GET', 'POST'])
def add_workout():
    form = WorkoutForm()
    if form.validate_on_submit():
        workout = Workout(title=form.title.data)
        db.session.add(workout)
        db.session.commit()
        return redirect(url_for('workouts'))
    return render_template('add_workout.html', form=form)

4. Установка и запуск

1. Установка зависимостей:
pip install -r requirements.txt

2. Инициализация БД:
flask db init
flask db migrate
flask db upgrade

3. Запуск приложения:
flask run
